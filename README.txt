How to start the web server: 

To start the web server you first need node.js, mongoDB and NPM installed. 
Then to start the web server you need to use command prompt to get into the file directory that has the backend file. 
From there you need to start the server by doing node server.js and verify that the server is connected.

Example of backend commands: 
cd ~/GymCompetitionApp/backend
node server.js

Second server:
There is a second server for react, to start this server you need to launch a different command prompt window then where the web server is. 
You then need to go into the directory that has the gym app and go into the frontend file. 
Then you do npm run dev, this will start the server, and there will be a link given. 
Copy and paste the link into your web browser, and you will be prompted to log in.

Example of frontend commands:
cd ~/GymCompetitionApp/frontend
npm run dev

How to navigate the web app:

The port is port 8080 for the back end, and for the frontend, the port is 5173. 
The link will be, http://localhost:5173/ . 
The admin username is admin, and the password is admin123. 
You can also register as a new user by clicking the register button underneath and then making a username and password of your choice.

MongoDB collections: 

The MongoDB collections are the Users, Communities, Goals, Activities, and UserGoalTargets
